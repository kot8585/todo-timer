# 프로젝트 실행 
```
npm run start
```

# 소개
- 할일을 기록하고 할일의 시간을 측정하고 기록하는 앱
- 카테고리 단위로 시간을 측정하는 다른 앱들과 다르게 세부단위인 할일의 시간을 측정할 수 있도록 차별점을 두었다.


# 기능 
- 투두 CRUD
- 타이머 측정, 중지(타이머가 중지되면서 시간만 추가됌), 완료(타이머가 멈추면서 시간추가와 투두도 완료되는 기능)
- 타임라인 CRUD
